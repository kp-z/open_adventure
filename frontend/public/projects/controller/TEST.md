## 已知问题

1. **浏览器限制**：无法直接执行 Shell 命令，只能复制到剪贴板
2. **HTTPS 要求**：Clipboard API 需要 HTTPS 或 localhost
3. **localStorage 限制**：配置仅保存在当前浏览器
