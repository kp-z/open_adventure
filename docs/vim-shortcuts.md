# Vim 快捷指令速查表

## 模式切换

| 快捷键 | 功能 |
|--------|------|
| `i` | 在光标前进入插入模式 |
| `I` | 在行首进入插入模式 |
| `a` | 在光标后进入插入模式 |
| `A` | 在行尾进入插入模式 |
| `o` | 在下方新建一行并进入插入模式 |
| `O` | 在上方新建一行并进入插入模式 |
| `Esc` | 退出插入模式，回到普通模式 |
| `v` | 进入可视模式（字符选择） |
| `V` | 进入可视行模式（行选择） |
| `Ctrl+v` | 进入可视块模式（列选择） |
| `:` | 进入命令行模式 |

## 移动光标

### 基本移动
| 快捷键 | 功能 |
|--------|------|
| `h` | 左移一个字符 |
| `j` | 下移一行 |
| `k` | 上移一行 |
| `l` | 右移一个字符 |
| `w` | 移动到下一个单词开头 |
| `W` | 移动到下一个单词开头（忽略标点） |
| `b` | 移动到上一个单词开头 |
| `B` | 移动到上一个单词开头（忽略标点） |
| `e` | 移动到当前单词结尾 |
| `E` | 移动到当前单词结尾（忽略标点） |
| `0` | 移动到行首 |
| `^` | 移动到行首第一个非空字符 |
| `$` | 移动到行尾 |
| `gg` | 移动到文件开头 |
| `G` | 移动到文件结尾 |
| `数字+G` | 移动到指定行（如 `10G` 移动到第10行） |
| `:数字` | 移动到指定行（如 `:10` 移动到第10行） |

### 屏幕移动

|--------|------|
| `Ctrl+f` | 向下翻一页 |
| `Ctrl+b` | 向上翻一页 |
| `Ctrl+d` | 向下翻半页 |
| `Ctrl+u` | 向上翻半页 |
| `H` | 移动到屏幕顶部 |
| `M` | 移动到屏幕中间 |
| `L` | 移动到屏幕底部 |
| `zz` | 将当前行移动到屏幕中央 |
| `zt` | 将当前行移动到屏幕顶部 |
| `zb` | 将当前行移动到屏幕底部 |

### 搜索移动
| 快捷键 | 功能 |
|--------|------|
| `/pattern` | 向下搜索 pattern |
| `?pattern` | 向上搜索 pattern |
| `n` | 跳转到下一个搜索结果 |
| `N` | 跳转到上一个搜索结果 |
| `*` | 向下搜索光标所在单词 |
| `#` | 向上搜索光标所在单词 |
| `f字符` | 在当前行向右查找字符 |
| `F字符` | 在当前行向左查找字符 |
| `t字符` | 在当前行向右查找字符（光标停在字符前） |
| `T字符` | 在当前行向左查找字符（光标停在字符后） |
| `;` | 重复上次 f/F/t/T 操作 |
| `,` | 反向重复上次 f/F/t/T 操作 |

|--------|------|
| `x` | 删除光标所在字符 |
| `X` | 删除光标前一个字符 |
| `dd` | 删除当前行 |
| `数字+dd` | 删除指定行数（如 `3dd` 删除3行） |
| `dw` | 删除到下一个单词开头 |
| `db` | 删除到上一个单词开头 |
| `d$` 或 `D` | 删除到行尾 |
| `d0` | 删除到行首 |
| `dG` | 删除到文件末尾 |
| `dgg` | 删除到文件开头 |
| `diw` | 删除当前单词（不包括空格） |

### 粘贴

|--------|------|
| `r字符` | 替换光标所在字符 |

## 组合技巧

### 常用组合
- `ci"` - 修改双引号内的内容
- `di(` - 删除括号内的内容
- `yap` - 复制当前段落
- `dap` - 删除当前段落
- `gUiw` - 将当前单词转为大写
- `guiw` - 将当前单词转为小写
- `ggVG` - 选中全文
- `ggdG` - 删除全文
- `:%y` - 复制全文

### 数字前缀
大多数命令都可以加数字前缀来重复执行：
- `3dd` - 删除3行
- `5j` - 向下移动5行
- `2w` - 向前移动2个单词
- `10x` - 删除10个字符

## 快速记忆技巧

- **d** = delete（删除）
- **y** = yank（复制）
- **p** = paste（粘贴）
- **c** = change（修改）
- **i** = inner（内部）
- **a** = around（周围）
- **w** = word（单词）
- **s** = sentence（句子）
- **p** = paragraph（段落）
- **t** = tag（标签）

## 进阶技巧

### 文本对象
- `iw` - 内部单词（不包括空格）
- `aw` - 整个单词（包括空格）
- `is` - 内部句子
- `as` - 整个句子
- `ip` - 内部段落
- `ap` - 整个段落
- `i"` - 双引号内
- `a"` - 双引号及内容
- `i(` 或 `ib` - 括号内
- `a(` 或 `ab` - 括号及内容
- `i{` 或 `iB` - 大括号内
- `a{` 或 `aB` - 大括号及内容
- `it` - HTML/XML 标签内
- `at` - HTML/XML 标签及内容

### 寄存器
- `"ay` - 复制到寄存器 a
- `"ap` - 从寄存器 a 粘贴
- `"+y` - 复制到系统剪贴板
- `"+p` - 从系统剪贴板粘贴
- `:reg` - 查看所有寄存器内容

---

**提示**：这些命令可以组合使用，形成强大的编辑能力。熟练掌握这些快捷键可以大大提高编辑效率！
