# 加载界面日志输出对比分析

## 概述
本文档对比分析应用初始化加载和游戏加载两个场景的日志输出实现。

## 1. 应用初始化加载（InitializationScreen）

### 位置
- 组件：`frontend/src/app/components/InitializationScreen.tsx`
- 上下文：`frontend/src/app/contexts/InitializationContext.tsx`

### 日志输出实现

#### ✅ 已实现的日志功能

**日志来源**：
```typescript

  isInitialized: checkInitialState(),
```

**日志内容**：

6. `✓ 初始化完成 (XXXms)`

**显示方式**：
```tsx

  <motion.div className="bg-white/5 rounded-xl border border-white/10 p-4 max-h-48 overflow-y-auto">

        <motion.div
          key={index}

```

**特点**：
- ✅ 显示详细的初始化步骤
- ✅ 显示每个步骤的数量（如扫描到的文件数）
- ✅ 显示总耗时
- ✅ 使用动画效果（逐条淡入）
- ✅ 自动滚动（最多显示 10 条）
- ✅ 使用等宽字体（font-mono）

## 6. 代码示例：游戏加载日志改进

### 完整实现代码

```typescript
// Microverse.tsx 中添加
const [loadingLogs, setLoadingLogs] = useState<string[]>([]);

const addLoadingLog = useCallback((message: string) => {
  setLoadingLogs(prev => [...prev.slice(-9), message]);
}, []);

// 在 useEffect 中使用
useEffect(() => {
  if (isGameCached()) {
    return;
  }

  const loadingSteps = [
    { text: '正在启动游戏模式...', delay: 0 },
    { text: '连接游戏服务器...', delay: 800 },
  ];

  loadingSteps.forEach(({ text, delay }) => {
    setTimeout(() => {
      setLoadingText(text);
      addLoadingLog(`✓ ${text}`);
    }, delay);
  });
}, [addLoadingLog]);

// 在消息监听中使用
const handleMessage = (event: MessageEvent) => {
  if (event.data?.type === 'godot-loading') {
    const { status, percentage } = event.data;

    switch (status) {
      case 'start':
        const startMsg = '初始化游戏引擎...';
        setLoadingText(startMsg);
        addLoadingLog(`✓ ${startMsg}`);
        break;
      case 'progress':
        const progressMsg = `加载游戏资源... ${percentage || 0}%`;
        setLoadingText(progressMsg);
        if (percentage % 20 === 0) { // 每 20% 记录一次
          addLoadingLog(`✓ ${progressMsg}`);
        }
        setLoadingProgress(percentage || 0);
        break;
      case 'complete':
        const completeMsg = '准备游戏画布...';
        setLoadingText(completeMsg);
        addLoadingLog(`✓ ${completeMsg}`);
        // ...
        break;
    }
  }
};
```
