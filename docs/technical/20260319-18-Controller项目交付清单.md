## 🚀 快速开始

### 1. 启动前端服务
```bash
cd frontend && npm run dev
```

### 2. 访问 Controller 页面
```
http://localhost:5173/projects/controller/
```

### 3. 配置快捷键
1. 右键点击任意按钮
2. 填写配置信息
3. 点击"保存"

### 4. 执行命令
1. 左键点击已配置的按钮
2. 命令自动复制到剪贴板
3. 粘贴到终端执行
