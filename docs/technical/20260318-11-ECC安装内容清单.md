## 🎯 核心特性

### 1. 不可变性原则 (CRITICAL)
- 永远创建新对象，永远不要修改现有对象
- 防止隐藏副作用
- 使调试更容易
- 支持安全并发

### 2. Hook 运行时控制
```bash
# 设置 Hook Profile
export ECC_HOOK_PROFILE=minimal    # 最小化
export ECC_HOOK_PROFILE=standard   # 标准（默认）
export ECC_HOOK_PROFILE=strict     # 严格

# 禁用特定 Hooks
export ECC_DISABLED_HOOKS=pre:bash:tmux-reminder,post:edit:format
```
