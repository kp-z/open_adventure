# Release Notes - v0.1.7

**发布日期**: 2026-03-02

## 本次更新重点

本次更新主要增强了启动脚本的灵活性，新增后台运行模式，让用户可以更方便地管理 Claude Manager 服务。

## ✨ 新增功能

- **后台运行模式**: `start.sh` 新增 `-d` / `--daemon` 参数，支持后台运行
  - 前端和后端都在后台运行，不受终端关闭影响
  - 即使 Ctrl+C 也不会停止服务
  - 日志自动输出到 `docs/logs/backend.log` 和 `docs/logs/frontend.log`

- **停止脚本**: 新增 `stop.sh` 脚本，用于优雅停止后台服务
  - 自动清理所有后台进程
  - 验证端口释放
  - 支持强制停止（如果进程无响应）

## 🔧 改进

- 改进了进程管理机制，新增 `FRONTEND_PID_FILE` 用于追踪前端进程
- 优化了启动脚本的输出信息，后台模式下显示更详细的进程信息
- 增强了进程清理逻辑，确保所有子进程都能被正确停止

## 使用方法

### 后台运行
```bash
./start.sh -d
# 或
./start.sh --daemon
```

### 停止后台服务
```bash
./stop.sh
```

### 前台运行（默认）
```bash
./start.sh
```
