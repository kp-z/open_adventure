# Open Adventure v0.1.4 发布说明

**发布日期**: 2026-02-27
**版本**: v0.1.4
**类型**: 功能改进版本

## 新增功能

### 1. 改进的启动脚本

**功能描述**:
start.sh 现在可以同时启动前端和后端服务，提供完整的开发体验。

**主要改进**:
- ✅ 同时启动前端（5173）和后端（8000）服务
- ✅ 后端在后台运行，前端在前台运行
- ✅ 主要入口显示为前端 http://localhost:5173
- ✅ 本地开发使用 ENV=development（启用 reload）
- ✅ 自动安装前端依赖（npm install）
- ✅ 按 Ctrl+C 时自动停止所有服务
- ✅ 后端日志输出到 docs/logs/backend.log

**启动信息**:
```
============================================
✅ Open Adventure is running!
============================================

🌐 Frontend: http://localhost:5173
🔧 Backend API: http://localhost:8000
📚 API Docs: http://localhost:8000/docs

Press Ctrl+C to stop all servers
============================================
```

## 安装说明

### 首次安装

1. 下载对应平台的发布包

2. 解压：
   ```bash
   tar -xzf open-adventure-v0.1.4-{platform}-{arch}.tar.gz
   cd open-adventure-v0.1.4-{platform}-{arch}
   ```

3. 运行启动脚本：
   ```bash
   ./start.sh
   ```

4. 首次运行会自动：
   - 创建 Python 虚拟环境
   - 安装 Python 依赖
   - 安装前端依赖（npm install）
   - 从 .env.example 创建 .env 文件

5. 编辑配置（如需要）：
   ```bash
   nano backend/.env
   ```

6. 访问应用：
   - 前端界面: http://localhost:5173
   - API 文档: http://localhost:8000/docs

### 从 v0.1.3 升级

如果你已经安装了 v0.1.3，建议重新安装：

1. 备份数据库（如果有重要数据）：
   ```bash
   cp backend/open_adventure.db ~/open_adventure_backup.db
   ```

2. 删除旧版本，按照上述"首次安装"步骤安装新版本

3. 恢复数据库：
   ```bash
   cp ~/open_adventure_backup.db backend/open_adventure.db
   ```
